About

"${name}" is a software that implements a number of tools for the creation and use of automatic text classifiers.

The software includes a web user interface and a web service API.

The public interface of the service publishes a set of selected classifiers.

Classification requests are limited to a maximum hourly rate. The counter for the limit is reset every hour.
