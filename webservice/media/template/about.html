About

"${name}"is software that implements a number of tools for the creation and use of automatic text classifiers.

The software includes a web client user interface and a web service API.

The demo of the software is limited to the use of a set of predefined classifier.

Classification requests are limited to a maximum hourly rate. The counter for the limit is reset every hour.

An authorization key with a higher limit can be given upon request. Keys may be available depending on the server load and other factors.
